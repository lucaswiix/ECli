# wiixzerECLI

WiixzerECLI is a javascript program in command line for calculate food orders from a group of people..

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install libraries.

```bash
npm install
```

## Usage

```javascript
cd ECli
node index.js
```
